# Competitive Landscape Visualizer

Map PurposeMed's competitive landscape on an interactive 2x2 matrix with customizable axes, bubble sizing, and per-brand filtering. Position competitors like Maple, TELUS Health, Cerebral, Done, and PlushCare against PurposeMed's three brands -- then click any bubble to add monitoring notes.

---

## Requirements

The lab brief describes a competitive analysis tool that plots competitors across customizable dimensions (Geographic Reach, Stigma Sensitivity, Patient Volume, etc.). It should support filtering by PurposeMed brand (Freddie, Frida, Foria), variable bubble sizing, and click-to-annotate functionality for tracking competitive intelligence.

![Competitive Landscape Visualizer requirements](/img/labs/claude-desktop/competitive-landscape-visualizer-requirements.png)

---

## Result

The artifact is a dark-themed bubble chart with dropdown filters for Brand, X-Axis, Y-Axis, and Bubble Size. Competitors appear as color-coded bubbles of varying sizes plotted on a 2x2 grid. PurposeMed's brands cluster in the high-reach, high-sensitivity quadrant, while competitors spread across the landscape. The interactive controls let you explore different strategic dimensions in real time.

## How to Recreate This

1. List your competitors with data points across the dimensions you care about
2. Open [Claude](https://claude.ai) and start a new conversation
3. Share your competitive data and ask for an interactive 2x2 bubble chart with customizable axes
4. Iterate -- add annotations, new competitors, or export-ready formatting

:::tip
Customizable axes make this tool reusable across different strategic questions -- swap Geographic Reach for Pricing, or Stigma Sensitivity for Clinical Depth. See [Research Workflows](/research-and-content/research-workflows) for how to use AI to gather the competitive data that feeds into this tool.
:::
