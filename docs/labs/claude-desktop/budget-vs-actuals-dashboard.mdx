# Budget vs. Actuals Dashboard

Track budget performance across all PurposeMed brands (Freddie, Frida, Foria) with a consolidated dashboard that compares planned allocations against actual spend. Filter by month and department to spot variances, identify departments over budget, and drill into detailed breakdowns.

---

## Requirements

The lab brief calls for a FY2026 financial dashboard with month and department filters. It should display total budget vs. actual, net variance, departments over budget, and include both bar chart comparisons and variance distribution visualizations. The tool should support drill-down by department.

![Budget vs. Actuals Dashboard requirements](/img/labs/claude-desktop/budget-vs-actuals-dashboard-requirements.png)

---

## Result

The artifact is a dark-themed financial analytics dashboard showing key metrics cards (Total Budget $1.29M, Total Actual $1.35M, Net Variance $64K, 2 Departments Over Budget). It includes a bar chart comparing budget vs. actual by department, a donut chart showing variance distribution across Clinical Ops, Engineering, Marketing, Pharmacy Ops, and G&A, plus a detailed variance breakdown table.
