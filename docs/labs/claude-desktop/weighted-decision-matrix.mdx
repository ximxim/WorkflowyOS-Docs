# Weighted Decision Matrix

Evaluate which new telehealth vertical PurposeMed should launch next using weighted criteria scoring and sensitivity analysis. Define criteria (Market Size, Regulatory Ease, Competitive Density, Mission Alignment, Operational Readiness), assign weights, score each option, and see how rankings shift when you adjust the top weight.

---

## Requirements

The lab brief calls for a group exercise where the team evaluates options (menopause care, chronic pain, dermatology) against weighted criteria. The tool should auto-calculate totals, rank options, and include sensitivity analysis showing how changing the top weight affects the winner. Designed as a collaborative, 25-minute exercise.

![Weighted Decision Matrix requirements](/img/labs/claude-desktop/weighted-decision-matrix-requirements.png)

---

## Result

The artifact features three tabs: Scoring Matrix, Results & Ranking, and Sensitivity Analysis. The Criteria & Weights section shows five adjustable criteria totaling 100%. A scoring grid lets you rate each vertical from 0 (poor) to 5 (excellent), with auto-calculated weighted totals. The interface is designed for live group collaboration with an easy difficulty rating.

## How to Recreate This

1. Define the decision you need to make and the options on the table
2. Open [Claude](https://claude.ai) and start a new conversation
3. List your evaluation criteria with weights and ask for an interactive decision matrix with sensitivity analysis
4. Iterate -- add more options, refine criteria definitions, or create a visual bar chart comparison

:::tip
Decision matrices are powerful for group exercises because they make trade-offs explicit. The sensitivity analysis tab reveals whether your conclusion is robust or depends heavily on one weight assumption. See [Executive Role in AI](/leadership/executive-role-in-ai) for more on using AI to support strategic decisions.
:::
